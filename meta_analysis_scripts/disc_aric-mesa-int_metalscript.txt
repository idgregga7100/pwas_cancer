#metal script to run analysis for ARIC studies

MARKER SeqId
ALLELE A1 A2  
PVALUE TWAS.P
EFFECT TWAS.Z
WEIGHTLABEL NSNP 
PROCESS ARIC_discovery_breast.txt

MARKER SeqId.x
ALLELE A1 A2  
PVALUE pvalue
EFFECT zscore
WEIGHTLABEL n_snps_used 
PROCESS TOPMed_discovery_breast.txt

MARKER SeqId.x
ALLELE A1 A2
PVALUE pvalue
EFFECT zscore
WEIGHTLABEL n_snps_used
SEPARATOR TAB
PROCESS INTERVAL_discovery_breast.txt

OUT METAL_ARIC-MESA-INT_disc_breast .tbl
ANALYZE

CLEAR
